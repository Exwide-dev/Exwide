# Exwide

使用 CPython 制作解释器的 Exwide 语言实现。

* * *

## 说明

**dev** 分支为开发分支，功能可能不稳定，请勿用于正式环境。该分支的特性仍在频繁变动中，因此 Exwide 的官方教学和文档内容将始终以 **main** 分支的最新版本为准。

当 **dev** 分支的功能确定后，将首先合并至 **rc** 分支，经过 Alpha 与 Beta 测试并修复相关问题后，再正式并入 **main** 分支。对于 **main** 分支的版本管理，小版本更新周期不超过半年，大版本更新需在改动幅度达到一定程度并经团队讨论通过后发布，日常的 bug 修复则仅通过修订号体现。
